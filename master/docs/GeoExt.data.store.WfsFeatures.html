(static) cacheFeatureCount

Cache the total number of features be queried from when the store is first loaded to use for the remaining life of the store. This uses resultType=hits to get the number of features and can improve performance rather than calculating on each request. It should be used for read-only layers, or when the server does not return the feature count on each request.
